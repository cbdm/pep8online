What Is This Website?

This is an online python code style checker modified for UCI's ICS 31 course.
Your code will be analyzed for style, so you can use this tool to check for mistakes beforehand.
Just paste your code or upload file and see results.

How Does It Work?

You can paste your code or upload a .py file to check for style.
This project was originally built by Valentin Bryukhanov, and the original project can be found here.

In order to use newer style guides, one of your TA's (Caio :) ported the code to python3 and changed the libraries being used to analyze the code.
This modified version can be found here.

Why Use It?

To help you produce code that's easier to understand.
