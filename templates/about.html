What Is It?

This is an online Pycodestyle checker modified for UCI's ICS 31 course.
Your code might be analyzed for style, so you can use this website to check beforehand.
Just paste your code or upload file and see results.

How Does It Work?

You can paste your code or upload a .py file to check for style.
This project was originally built by Valentin Bryukhanov, and the original project can be found here.

In order to use newer style guides, one of your TA's (Caio :) ported the code to python3 and changed pep8 to pycodestyle instead.
This modified version can be found here.

Why?

To encourage students to produce code that's easier to understand.
